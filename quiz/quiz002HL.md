# Quiz 002 HL
Given 2 numbers A and B, Output TRUE if one of themis 20 or if their sum is 20.
HL: You recieve two lists of numbers A, and B

```.py
A_list = [10, 30, 10, 26]
B_list = [20, 15, 5, -6]

Output = False

for a in A_list:
    if a == 20:
        Output = True
for b in B_list:
    if b == 20:
        Output = True

print(f"The answer to the quiz 002 is {Output}")
```
