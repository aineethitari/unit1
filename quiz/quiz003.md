# Quiz 003
Create a program that translate the proteins in the DNA chain as shown below

```.py
in_protein = str(input("Please enter protein"))
if in_protein == "A":
    Output = "T"
elif in_protein == "G":
    Output = "C"
elif in_protein == "T":
    Output = "A"
elif in_protein == "C":
    Output = "G"
print(f"The translation of the protein is {Output}")
```
